Release Notes

Installation
-----------------------------------------
java runtime is required, minimully 1.3, however 1.4 is recommended. Java runtime for most
platforms available available from http://java.sun.com/j2se/ and various
locations.

Decompress distribution files either with winzip or other compression utility. Directory
created can be placed at any location as application is self contained.

Browse to 'edi4j' directory and in first level lie 2 start scripts. On windows double click
start.bat, on unix, linux, or macintosh OSX execute start.sh
